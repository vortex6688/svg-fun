Borders

Use border utilities to quickly style the border and border-radius of an element. Great for images, buttons, or any other element.

Border

Add classes to an element to remove all borders or some borders.

<span class="border-0"></span>
<span class="border-top-0"></span>
<span class="border-right-0"></span>
<span class="border-bottom-0"></span>
<span class="border-left-0"></span>

Border-radius

TypeNetwork does not use border radiuses. The standard border radius classes are disabled, except rounded-circle which is used for avatar images.
