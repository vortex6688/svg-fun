Modal

Modals are streamlined, but flexible dialog prompts powered by JavaScript. They support a number of use cases from user notification to completely custom content and feature a handful of helpful subcomponents, sizes, and more.

NgbModal

A service to open modal windows. Creating a modal is straightforward: create a template and pass it as an argument to the "open" method!

Methods

open
Signature: open(content: any, options: NgbModalOptions)
Return type: NgbModalRef
Opens a new modal window with the specified content and using supplied options. Content can be provided as a TemplateRef or a component type. If you pass a component type as content than instances of those components can be injected with an instance of the NgbActiveModal class. You can use methods on the NgbActiveModal class to close / dismiss modals from "inside" of a component.

NgbModalOptions

Represent options available when opening new modal windows.

Properties

backdrop
Type: boolean | "static"
Whether a backdrop element should be created for a given modal (true by default). Alternatively, specify 'static' for a backdrop which doesn't close the modal on click.
keyboard
Type: boolean
Whether to close the modal when escape key is pressed (true by default).
size
Type: "sm" | "lg"
Size of a new modal window.
windowClass
Type: string
Custom class to append to the modal window

NgbModalRef

A reference to a newly opened modal.

Properties

componentInstance
Type: any
The instance of component used as modal's content. Undefined when a TemplateRef is used as modal's content.
result
Type: any
A promise that is resolved when a modal is closed and rejected when a modal is dismissed.

Methods

close
Signature: close(result: any)
Return type: void
Can be used to close a modal, passing an optional result.
dismiss
Signature: dismiss(reason: any)
Return type: void
Can be used to dismiss a modal, passing an optional reason.

NgbActiveModal

A reference to an active (currently opened) modal. Instances of this class can be injected into components passed as modal content.

Methods

close
Signature: close(result: any)
Return type: void
Can be used to close a modal, passing an optional result.
dismiss
Signature: dismiss(reason: any)
Return type: void
Can be used to dismiss a modal, passing an optional reason.
Heads up! The NgbModal service needs a container element with the ngbModalContainer directive. The ngbModalContainer directive marks the place in the DOM where modals are opened. Be sure to add <template ngbModalContainer></template> somewhere under your application root element.

Using the grid

Utilize the Bootstrap grid system within a modal by nesting .container-fluid within the .modal-body. Then, use the normal grid system classes as you would anywhere else.
