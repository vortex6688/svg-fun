Dropdown

The Bootstrap dropdown plugin allows you to toggle content on your pages

Dropdown Example

Dropdown Button Example / Single Option Select

  • Item 1
  • Another Item
  • Some Other Item 
<div class="row">
  <div class="col">
    <div ngbDropdown class="d-inline-block">
      <button style="min-width: 15rem;" class="btn-secondary text-left" id="dropdownSingleOption" ngbDropdownToggle>
        <span class="h6">Toggle Dropdown</span>
        <icon-select class="icon"></icon-select>
      </button>
      <ul class="dropdown-menu list-group-flush" aria-labelledby="dropdownSingleOption">
        <li class="list-group-item pl-4">
          <icon-close class="icon"></icon-close>
          <span>Item 1</span>
        </li>
        <li class="list-group-item pl-4">
          <span>Another Item</span>
        </li>
        <li class="list-group-item pl-4">
          <span>Some Other Item</span>
        </li>
      </ul>
    </div>
  </div>
